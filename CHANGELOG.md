## [Unreleased]

### Added

- Initial release of Open Docs.
- Marketing homepage: hero, features, benefits, philosophy, community, and CTA.
- Documentation system: runtime MDX, nested routes, `meta.json` ordering,
  frontmatter, breadcrumbs, previous/next navigation, and last-updated metadata.
- Sticky table of contents with an animated active-section indicator.
- Sidebar with collapse persistence and an accessible mobile drawer.
- Command-palette search (`⌘K` / `Ctrl K`) backed by a static Fuse.js index.
- MDX component library: Callout, Cards, Tabs, Steps, CodeBlock, FileTree, Badge.
- Dark mode with no flash on load and a brand color system (50–950).
- Copy-as-markdown and edit-this-page actions on every doc.
- Static export build for zero-runtime hosting (GitHub Pages ready).
- Test suite with 90%+ coverage on the content/navigation pipeline.
