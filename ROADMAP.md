# Roadmap

This roadmap captures the direction of Open Docs. It is intentionally
high-level; issues and discussions track the details. Priorities may shift based
on community feedback.

## ✅ Shipped

- Static-export documentation platform (Next.js App Router + MDX)
- Marketing homepage and full docs shell
- Command-palette fuzzy search (Fuse.js, static index)
- MDX component library (Callout, Cards, Tabs, Steps, CodeBlock, FileTree, Badge)
- Dark mode, brand color system, and accessibility baseline (WCAG AA)
- Test suite with 90%+ coverage and CI

## 🛠️ In progress

- `create-open-docs` CLI generator
- Release automation with Changesets
- GitHub Pages deployment workflow

## 🔭 Planned

- Versioned documentation support
- i18n / multi-language content
- Built-in OG image generation per page
- Algolia / Pagefind search adapter as an alternative to Fuse.js
- Light/dark code themes configurable via brand tokens
- More starter content templates (API reference, changelog, blog)

## 💡 Under consideration

- Visual theme editor
- MDX component marketplace
- Analytics adapters (privacy-friendly by default)
