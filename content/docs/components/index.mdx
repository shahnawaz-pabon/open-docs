# Components

Open Docs ships a small, focused library of MDX components. They are registered
globally, so you can use them in any `.mdx` file **without importing anything**.

<Callout type="tip" title="Source-code ownership">
  Every component lives in `components/mdx/` as plain, readable TSX. Fork it, restyle it, or delete
  what you don't need.
</Callout>

## Registering your own

All components are wired up in `components/mdx/index.ts`. To add one, export it
from that file's `customMdxComponents` map and it becomes available everywhere.
