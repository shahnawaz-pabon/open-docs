# Guides

These guides walk through the most common things you'll do with Open Docs once your
project is running.

- **[Writing Content](/docs/guides/writing-content)** — frontmatter, folders, and `meta.json`.
- **[Deploying](/docs/guides/deploying)** — ship to GitHub Pages or any static host.

Each guide is short and task-focused. If something is missing, contributions are very
welcome.
