# Introduction

**Open Docs** is a production-grade, MIT-licensed documentation template for modern
TypeScript projects. It is designed to be the definitive starting point for teams who
want world-class docs without adopting a heavyweight framework.

There is no documentation engine to learn and no vendor lock-in. Every component is plain
Next.js, React, and TypeScript that you own and can read top to bottom.

## Why Open Docs

- **Zero framework lock-in** — it's just Next.js App Router and MDX.
- **Source-code ownership** — fork it, read it, change anything.
- **Accessible & fast** — WCAG AA, Lighthouse 95+, static export ready.
- **AI-assistant friendly** — a documented architecture that tools understand.

## What's included

A marketing homepage, a full docs system with sidebar navigation, breadcrumbs, a sticky
table of contents, fuzzy command-palette search, a polished MDX component library, and a
complete testing and CI setup.
